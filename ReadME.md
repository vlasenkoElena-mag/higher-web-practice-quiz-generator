# quizzer

Проект для создания и прохождения небольших викторин в браузере.

 - **GitHub:** (https://github.com/vlasenkoElena-mag/higher-web-practice-quiz-generator)

Описание
Этот репозиторий содержит одностраничное веб-приложение для генерации викторин, их просмотра и прохождения. Приложение поддерживает создание вопросов с вариантами ответов, локальное хранение викторин и отображение результатов игры.

Быстрый старт
1. Установите зависимости:

```
npm install
```

2. Запустить режим разработки (откроется в браузере):

```
npm run dev
```

3. Собрать продакшн-билд:

```
npm run build
```

4. Для локального предпросмотра собранной версии можно использовать любой статический сервер или `vite preview`:

```
npm run preview
```

Доступные скрипты
- `dev`: запускает Vite в режиме разработки.
- `build`: собирает проект для продакшена.
- `lint`: запускает ESLint по проекту.
- `lint:fix`: запускает ESLint с автофиксами.

Структура проекта (основное)
- `index.html`, `quizzes.html`, `quiz.html` — точки входа для страниц.
- `src/` — исходный код приложения:
  - `components/`, `view/` — UI-компоненты и представления.
  - `presenters/` — логика взаимодействия между моделью и представлением.
  - `models/` — модели данных викторины и игрового процесса.
  - `utils/` — утилиты (хранение, валидация, DOM-helpers, события).
  - `pages/` — точечные файлы запуска для страниц (TypeScript).
- `assets/` и `css/` — стили и статические ресурсы.

Используемые библиотеки
- `zod` — схемы и валидация данных.
- `idb` — удобная обёртка для IndexedDB.
- `nanoid` — генерация уникальных идентификаторов.

Конфигурация и типы
- `vite.config.js` — конфигурация сборки и алиасы путей.
- `tsconfig.json` — настройки компилятора TypeScript.
